## Étapes d’exécution du Backend (.NET 4.8)

1. **Cloner le projet :**

   ```bash
   https://github.com/KlaiRayen/Softt365_Test_back
   ```

2. **Ouvrir la solution `.sln` dans Visual Studio.**

3. **Restaurer les packages NuGet** (Visual Studio le fait automatiquement à l’ouverture).

4. **Lancer l’application :**

**L’API sera disponible à l’adresse :**

   ```
   https://localhost:44365/
   ```

**La documentation Swagger est accessible à :**

   ```
   https://localhost:44365/swagger
   ```
